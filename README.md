# The Wild Oasis Hotel
## Intoduction 
Developed a hotel system dashboard that enables hotel employees to efficiently manage their hotel operations.
### Key features :
<ol>
  <li><b>Authentication</b> -> Allows hotel employees to securely log in and access the system</li>
  <li><b> Authorization</b> -> nsures users have the right permissions based on their roles</li>
  <li><b> CRUD operations for cabins</b> -> Enables employees to create, read, update, and delete cabin.</li>
  <li><b> CRD operations for bookings</b> -> Allows staff to create, read, and delete hotel bookings.</li>
  <li><b> update user data and password</b> -> Lets users modify their personal details and change passwords.</li>
  <li><b> settings updates</b> -> Provides options to customize system's data.</li>
  <li><b> charts</b> -> Displays hotel performance and numbers through interactive charts.</li>
  <li><b> create user </b>-> Allows admins to add new employees</li>
  <li><b> Dark Mode </b>-> Offers a user-friendly dark theme for better visual comfort.</li>
</ol>

## Technical skills
 #### Frontend : React.js, React Query, Styled Components, Recharts, React Hook Form.

# The Wild oasis Hotel
## Intoduction 
Developed a hotel system dashboard that enables hotel employees to efficiently manage their hotel operations.
### Key features :
<ol>
  <li> Authentication</li>
  <li> Authorization</li>
  <li> CRUD operations for cabins</li>
  <li> CRD operations for bookings</li>
  <li> update user data and password</li>
  <li> settings updates</li>
  <li> statistics data visualization with charts</li>
  <li> create user and Dark Mode</li>
</ol>

## Teachnical skills
 #### Frontend : React.js, React Query, Styled Components, Recharts, React Hook Form.
 #### Backend : supabase
